快速入门 #

第一步、安装好 node 环境 #

点击去 node 中文网下载,傻瓜式安装,一直下一步就行(Windows 默认会安装到 C 盘,最好是不改路径)

用命令行检查 node 和 npm 是不是装好了

node -v
npm -v

第二步、安装 pnpm(感觉这个安装依赖的界面比较好看) #

npm i pnpm -g

第三步、使用 vite 初始化 react 项目 #

# 使用vite初始化react项目,选择react模板(会ts也可以选ts,我这里选择js)
pnpm create @vitejs/app demo
# 进入demo目录
cd demo
# 安装依赖
pnpm i
# 运行项目
pnpm dev

打开浏览器访问 http://localhost:3000/

第四步、使用 vscode(其他编辑器也行,我比较喜欢这个)打开 demo 目录 #

修改一下 vite.config.js

import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";

export default defineConfig({
  plugins: [react()],
  server: {
    // 可以指定端口号
    port: 3000,
    // 启动时打开默认浏览器
    open: true,
  },
});

停掉刚刚跑起来的项目,重新执行一下

pnpm dev

这时候,vite 就会帮你自动打开默认浏览器访问 http://localhost:3000/

修改一下 index.html 的编码,不然 chrome 总是跳翻译

<!DOCTYPE html>
<html lang="zh">
  <head>
    <meta charset="UTF-8" />
    <link rel="icon" type="image/svg+xml" href="/src/favicon.svg" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Vite App</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/main.jsx"></script>
  </body>
</html>

清理一下默认模板,删了 src 文件夹下除 main.jsx 之外的所有文件,修改 mian.jsx 如下

import React from "react";
import ReactDOM from "react-dom";
// 引入App组件
import App from "./App";

ReactDOM.render(<App />, document.getElementById("root"));

在 src 目录下创建第一个组件 App.jsx

import React from "react";

export default () => {
  return <h1>hello world</h1>;
};

刷新浏览器页面,到此入门 demo 完成
